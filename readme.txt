== Description ==

This program is designed for designers and coders who not good at wordpress.
<br/> 
How to create a new carousel module in the PondolPlugin Carousel below. 
<br/>
- Create a new folder in the folder pondolplugin-carousel/modules. 
<br/>
- a module will be displayed in skins tab.
<br/> 
- A new folder must have carousel.php file and if you want to a skin icon on "skins tab" put thumb.png file in the folder
<br/> 
- Each module is independent. so you can make carousel modules you want easily.
<br/> 
- just look at existing carosel module, "default". that is how to make. 
<br/>

== Installation ==
1. extract pondolplugin-carousel.zip and Upload  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
